A small demo website for a web development agency. I built it to try out a few
AceternityUI components and see how it works.

- Static site with a few pages
- Smooth, fun animations using AceternityUI
- Simple contact form at the end
- Form handled by Web3Forms and server actions

Tech: Next.js, TypeScript, Tailwind CSS, AceternityUI, Web3Forms.
